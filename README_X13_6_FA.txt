DeutschWeg X13.6 — Cumulative Entry, Universal Categories & Data-Safe Update Base

این بسته سورس تجمعی وب برای GitHub Pages است و بر X13.4 ساخته شده است.

موارد اضافه‌شده:
- ورودی ۷ مرحله‌ای Mobile-first با الهام معماری PflegeDeutsch Pro، ولی با محتوای عمومی زبان آلمانی.
- سطح فعلی، سطح هدف، هدف، زمان، روزهای مطالعه، اولویت مهارت، کتگوری‌ها و حوزه شغلی.
- ۱۱ کتگوری مادر که همه ۳۰ حوزه و ۳۴۱ موضوع موجود را پوشش می‌دهد.
- صفحه ورودی فشرده موبایل، Smart Shelves، دسترسی سریع و Category Drawer.
- حفظ Storage Key موجود: deutschweg_x12_user_data.
- Migration و Backup قبلی حفظ شده است.
- Mirror تکمیلی state در IndexedDB اضافه شده است.
- در Android باید applicationId=com.deutschweg.app و Origin داخلی appassets ثابت بماند.
