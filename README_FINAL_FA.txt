DeutschWeg v8.0 — Final Release Candidate

این بسته نسخه نهایی فنی/محصولی آماده انتشار GitHub Pages است.

ویژگی‌های قفل‌شده:
- Mobile-first PWA برای iPhone و Android
- تشخیص پوسته iOS / Android / Web
- Bottom Navigation پنج‌بخشی و حالت Navigation Rail در نمایشگر بزرگ
- Safe Area و Edge-to-Edge سازگار
- برنامه‌ریزی شخصی A1 تا C2
- Today Flow، Focus/Pomodoro و Learning Analytics
- Vocabulary + German TTS + Duden links
- Podcast Hub و Resource Router
- Global Search در واژگان، منابع، مهارت‌ها و روش‌ها
- Notification/Guidance Center داخلی
- Offline indicator و Service Worker بهبود‌یافته
- PWA Install guidance برای iOS/Android
- Accessibility: اندازه متن، کنتراست بالا، کاهش حرکت
- Safe Migration و Backup قبل از تغییر Schema
- Storage Key ثابت: deutschweg_user_data
- Data Schema: 14
- کانال‌ها فقط تبلیغ/معرفی هستند و در Learning Engine استفاده نمی‌شوند.

انتشار GitHub Pages:
1) همه فایل‌های داخل ZIP را در ریشه Repository قرار دهید.
2) config.js را فقط در صورت استفاده از Supabase تنظیم کنید.
3) هرگز Service Role Key را در GitHub قرار ندهید.
4) GitHub Pages را از main / root فعال کنید.
5) پس از انتشار، QR نهایی را از URL واقعی Pages بسازید.

وضعیت Final:
- Static/Syntax/Package QA انجام شده است.
- تست نهایی پذیرش باید روی URL واقعی HTTPS و حداقل یک iPhone و یک Android انجام شود.
- تا پایان تست واقعی دستگاه، برچسب مهندسی دقیق این بسته Final Release Candidate است؛ پس از PASS تست دستگاه همان بسته می‌تواند Final Release شود.
