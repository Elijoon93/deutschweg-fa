# DeutschWeg X13.6.2 — Web Navigation Parity

این نسخه فقط برای GitHub Pages/Web است و معماری منو را با الگوی فایل مرجع PflegeDeutsch هماهنگ می‌کند، بدون انتقال محتوای پزشکی.

## تغییرات
- Bottom Navigation: خانه، برنامه من، یادگیری، پیشرفت، بیشتر
- Home Smart Shelves: ادامه مسیر، پیشنهادها، جهان‌های منتخب، اخیراً استفاده‌شده
- Learn Center فشرده با Progressive Disclosure
- Context Navigation در بخش یادگیری: نمای کلی، موضوع‌ها، واژگان، تمرین، جست‌وجو
- More App Drawer با Search، Category Chips، Favorites، Recent و All Apps
- اتصال ۱۱ کتگوری مادر به ۳۰ حوزه و ۳۴۱ موضوع موجود
- حفظ Storage Key: `deutschweg_x12_user_data`
- بدون تولید یا افزایش مصنوعی واژه، جمله، ترجمه یا سطح CEFR

## انتشار
محتویات این پوشه را در ریشه Repository GitHub Pages قرار دهید. فایل `index.html` باید در Root باشد.
