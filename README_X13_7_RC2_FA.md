# DeutschWeg X13.7 RC2 — Web Finalization

این نسخه قابلیت آموزشی پراکنده جدید اضافه نمی‌کند. تمرکز فقط روی نهایی‌سازی نسخه وب است:

- رفع بازنشانی سطح هدف هنگام پایان ورودی هفت‌مرحله‌ای؛
- جلوگیری از کاهش اشتباه `appVersion` توسط لایه قدیمی X13.6؛
- تست خودکار Chromium برای عرض صفحه، ورودی، پنج مقصد اصلی، ۲۱ ابزار، حفظ داده و تفکیک اصالت محتوا؛
- حفظ کلید `deutschweg_x12_user_data` و Data Schema 1؛
- باقی‌ماندن پذیرش گوشی فیزیکی به‌صورت دستی و بدون ادعای PASS خودکار.

وضعیت: `Release Candidate 2`. نتیجه GitHub Actions و آزمون گوشی واقعی باید هر دو بررسی شوند.
