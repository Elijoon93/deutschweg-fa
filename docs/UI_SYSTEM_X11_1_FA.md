# DeutschWeg X11.1 — Signature Visual System

معیار رسمی طراحی:

- Duolingo-level usability
- Babbel-level seriousness
- Linear-level visual polish

## تغییرات این بازسازی

1. Design Tokens جدید با Signature Light/Dark.
2. Typography tokenized برای فارسی و لاتین.
3. Button hierarchy: Primary / Secondary / Tertiary / Danger.
4. چهار Card type: Learning / Progress / Action / Resource.
5. iOS Floating Frosted Navigation.
6. Android Material-like solid Navigation.
7. Desktop Navigation Rail.
8. Shadow فقط برای عناصر واقعاً floating.
9. حذف Version Labelهای قدیمی از UI.
10. Home با Hero + Continue CTA + Today + Recommendation + Progress + Quick Access.
11. Learn با پنج دسته اصلی و Secondary Sub-navigation.

این فایل فقط شرح Design System است؛ منبع اجرایی x11-ui.css و index.html هستند.
