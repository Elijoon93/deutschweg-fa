# DeutschWeg X11 — Global Product UI

معیار طراحی:

- Duolingo-level usability: شروع سریع، مسیر روشن، بازخورد فوری و کاهش اصطکاک.
- Babbel-level seriousness: محتوای آموزشی جدی، CEFR، آزمون و ساختار آکادمیک.
- Linear-level visual polish: مینیمالیسم، سلسله‌مراتب بصری، فاصله‌گذاری و تعامل دقیق.

## اصول قفل‌شده

1. Home مسیر یادگیری است، نه فهرست امکانات.
2. در هر صفحه فقط یک CTA اصلی غالب وجود دارد.
3. ابزارهای پیشرفته در More و Contextual Actions قرار می‌گیرند.
4. Bottom Navigation موبایل پنج مقصد ثابت دارد: Home / Today / Learn / Progress / More.
5. iOS از Floating Frosted Bar، Android از Surface Bar و Desktop از Navigation Rail استفاده می‌کند.
6. Shadow فقط برای سطوح شناور استفاده می‌شود؛ کارت‌های عادی Flat هستند.
7. Teal هویت اصلی برند است؛ Cyan برای Accent و Gold برای Achievement/Premium.
8. هیچ Asset یا هویت بصری سرویس‌های دیگر کپی نشده است.

## Information architecture

Home: Continue Learning → Today → Smart Recommendation → Quick Access → Metrics → Focus
Today: Daily timeline + Focus + Notes
Learn: Path / Exercise / Vocabulary / Listening / Pronunciation / Review
Progress: Overview / Time / Weeks / Exams
More: Planner / Calendar / Methods / Resources / Media / Persian Guide / Search / Notifications / Settings
